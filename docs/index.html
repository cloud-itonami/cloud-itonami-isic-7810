この紹介デスクが保証すること

  • マッチング基準に保護属性が使われた candidacy は載らない(公正性 — 均等法5条 / Title VII / Equality Act / AGG)
  • 紹介手数料は「年収 × 手数料率」の独立再計算と常に一致する
  • 就労資格の確認を要する候補者は、未確認のまま配置されない(入管法 / I-9 / right-to-work / AufenthG §4a)
  • マッチングも配置も、どの phase でも自動実行されない — 常に人間の承認
  • すべての決定が追記専用の監査台帳に残る
